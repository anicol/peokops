## Common Questions

### **"What if managers don't have cell service?"**
The link works for 24 hours, so they can use WiFi later. We also support email as a backup delivery method.

### **"What if someone forwards the link?"**
Links are tied to the run, not a user. If a manager forwards it to a shift supervisor, that's fineâ€”they can complete the checks together. You still get full visibility.

### **"Can managers save the link for tomorrow?"**
No. Each day gets a fresh link. Yesterday's link automatically expires at midnight.

### **"What about other features in PeakOps?"**
Passwordless is currently for daily micro-checks. For full dashboard access (reports, management, settings), we support traditional login or SSO. Best of both worlds.

### **"How much does it cost?"**
SMS costs are ~$0.01 per message. For a 40-location chain with daily checks: about $25/month. That's less than one hour of manager time saved from password resets.
